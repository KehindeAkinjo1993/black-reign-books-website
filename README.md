Tech stack (HTML, CSS, a bit of JS)
This project is a front-end implementation of the Black Reign Books publishing platform, developed from a full Figma design system.
Features include:
	•	Accessible navigation with scroll-activated header state
	•	Hero section with JavaScript typing animation
	•	Custom geometric pattern background built in Affinity Designer
	•	Grid-based service cards with icons (Boxicons)
	•	File-upload UI with drag-and-drop styling
	•	Responsive two-column layouts using CSS Grid & Flexbox
	•	Contact form and sponsor form
	•	Footer with structured link groups

The site was built with clean semantic HTML, responsive CSS, and progressive enhancement using vanilla JavaScript.
